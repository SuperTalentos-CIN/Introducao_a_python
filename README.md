# Introdução a Python - SuperTalentos CIn 

Este repositório contém os códigos e exemplos práticos da **Aula 1** da trilha SuperTalentos do Centro de Informática (CIn-UFPE). O material foca nos fundamentos essenciais da linguagem Python para quem está iniciando na programação.

## 📚 Tópicos Coordenados

Nesta aula, exploramos os seguintes pilares:

* **Manipulação de Dados:** Conversão entre tipos (`int` vs `str`) e operações básicas de soma.
* **Interação com Usuário:** Como utilizar a função `input()` para capturar dados.
* **Operadores e Condições:** Uso de operadores de comparação e estruturas condicionais simplificadas (*ternárias*).
* **Funções Matemáticas:** Utilização de funções built-in como `abs()`, `round()`, `max()` e `min()`.
* **Manipulação de Strings:** Uma biblioteca completa de métodos de string (como `.upper()`, `.split()`, `.replace()`, etc.) para tratar e formatar textos.

## 📂 Conteúdo dos Arquivos (`Python_archives`)

* `input.py`: Exemplo de entrada de dados do usuário.
* `soma.py`: Operações de incremento de variáveis.
* `igual.py`: Comparação de tipos e valores em Python.
* `maior_que.py`: Uso de estrutura condicional em uma única linha.
* `inteiro_e_string.py`: Verificação de tipos com a função `type()`.
* `python_math.py`: Exemplos de funções matemáticas essenciais.
* `python_string_methods.py`: Guia de referência rápida para métodos de manipulação de strings.
